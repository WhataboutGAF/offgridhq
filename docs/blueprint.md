# **App Name**: OFFGRID HQ

## Core Features:

- Showcase Hero Projects: Dynamic display of 5 'floating' project cards in the hero section, featuring illustrations/icons, project names, and interactive visual elements.
- Detailed Project Listings: A dedicated 'Our Projects' section to display detailed information for 3 key projects, including descriptions, thumbnail illustrations, and arrow button calls to action.
- AI Project Navigator: An interactive tool that intelligently guides visitors to discover which OFFGRID HQ project best suits their needs based on simple textual input, showcasing the company's AI capabilities.
- Brand Information Display: Comprehensive display of company mission, key statistics (e.g., 5+ PROJECTS, GLOBAL USERS), motivational text, and minimal footer details.
- Call to Action System: Clear and engaging calls-to-action throughout the page, including a prominent 'START EXPLORING' button in the navigation and 'Explore Projects' buttons.
- Responsive Design: Ensures a seamless viewing experience across various devices, adapting layout for mobile (e.g., stacked cards, hamburger menu) while maintaining generous spacing and design aesthetics.

## Style Guidelines:

- Primary color: Electric Blue (#4C5BFF). This vibrant blue embodies a modern, tech-savvy, and professional brand identity, suitable for key interactive elements.
- Background color: Light Warm Gray (#F6F5F2). This soft, almost neutral base provides a clean, calming backdrop that allows the brighter accent colors to stand out, supporting a minimalist SaaS aesthetic.
- Accent color: Lime Green (#C6FF2E). This energetic and playful green provides a striking contrast to the primary blue, perfect for calls to action, highlights, and adding a youthful, innovative touch.
- Headings: 'Poppins' (sans-serif) for its modern, precise, and contemporary feel. Body text: 'Inter' (sans-serif) for its readability and clean, neutral appearance. Note: currently only Google Fonts are supported.
- Utilize clean, modern, and slightly rounded SVG icons and custom illustrations for project cards and UI elements, reflecting a tech-savvy yet approachable brand. Decorative star icons will add a playful touch in specific sections.
- A minimalist and spacious layout featuring generous whitespace, soft drop shadows, and consistently rounded corners (12px-20px radius) for a modern SaaS aesthetic. Hero content is center-aligned, with interactive elements organized within a clean spacing grid. The design ensures mobile responsiveness with vertically stacked hero and project cards, and a hamburger menu for navigation.
- Subtle micro-animations will enhance user interaction, including a 'hover lift' effect on interactive cards, a gentle glow effect for buttons on hover, and a 'subtle floating' motion for the hero project cards to add visual depth and dynamism.
